# ChainerPruner

ChainerPruner: Channel Pruning framework for [Chainer](https://github.com/chainer/chainer)

## Installation

- requirements:
    - chainer >= 5.1.0
    - networkx
    - (optional) scipy


```bash
$ pip install git+https://github.com/DeNA/ChainerPruner
```
